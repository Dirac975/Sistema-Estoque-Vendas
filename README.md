Sistema de Estoque e Vendas

Sistema desenvolvido em Python com foco no aprendizado de Programação Orientada a Objetos (POO) e lógica de programação. O projeto permite realizar o cadastro de mercadorias, controle de estoque, simulação de vendas e acompanhamento do faturamento da loja.

Funcionalidades
Cadastro de produtos com nome, categoria, preço e quantidade em estoque.
Consulta de todos os produtos cadastrados.
Simulação de vendas com atualização automática do estoque.
Controle do faturamento total acumulado.
Relatório comercial contendo informações de estoque e receita.
Objetivos do Projeto

Este projeto foi criado para praticar conceitos fundamentais de Python, incluindo:

Classes e objetos.
Métodos e atributos.
Funções.
Estruturas condicionais.
Estruturas de repetição.
Manipulação de listas.
Organização de sistemas em módulos.
